### 섹션 2: 뷰락쿠란? (제품 소개 A)

- 제품 이미지 + 텍스트 (좌우 split 또는 카드형)
- 브랜드: 코칸도 (皇漢堂製薬 / Kokando)
- 제품명: 뷰락쿠A (ビューラックA)
- 주성분: 비사코딜 (Bisacodyl) 5mg/정
- 작용: 대장을 직접 자극하여 장의 연동운동을 촉진
- 효과 시간: 복용 후 6~11시간
- 특징:
  - 가성비가 좋음 (대용량 400정 가능)
  - 코락쿠 대비 부드러운 사용감
  - 소형 분홍색 정제, 삼키기 쉬움
- 라인업:
  - 뷰락쿠A 50정
  - 뷰락쿠A 100정
  - 뷰락쿠A 250정
  - 뷰락쿠A 400정
  - 뷰락쿠 소프트 (피코술페이트나트륨 - 더 순한 타입)

### 섹션 3: 코락쿠란? (제품 소개 B)

- 제품 이미지 + 텍스트
- 브랜드: 타이쇼 제약 (大正製薬 / Taisho Pharmaceutical)
- 제품명: 코락쿠 (コーラック / Colac)
- 주성분: 비사코딜 (Bisacodyl) 5mg/정
- 작용: 대장을 직접 자극하여 장의 연동운동을 촉진
- 효과 시간: 복용 후 6~11시간
- 특징:
  - 일본 국민 변비약 (인지도 1위)
  - 5층 코팅 기술 - 위에서 녹지 않고 장까지 도달
  - 다양한 라인업으로 증상별 선택 가능
- 라인업:
  - 코락쿠 (기본형) - 비사코딜 5mg
  - 코락쿠II - 비사코딜 5mg + DSS(변 연화제) 8mg
  - 코락쿠 퍼스트 - 비사코딜 2.5mg (초보자용, 절반 용량)
  - 코락쿠 허브 - 센노사이드 + 감초 (생약 성분)
  - 코락쿠Mg - 산화마그네슘 (비자극성, 순한 타입)
  - 코락쿠 좌약 타입 - 직장 삽입형 (즉효)
